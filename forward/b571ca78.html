SQLдҪңз”Ё

SQL з”ЁжқҘз®ЎзҗҶе’Ңж“ҚдҪңMySQLеҶ…йғЁзҡ„еҜ№иұЎ
SQLеҜ№иұЎпјҡ

  • еә“пјҡеә“еҗҚпјҢеә“еұһжҖ§
  • иЎЁпјҡиЎЁеҗҚпјҢиЎЁеұһжҖ§пјҢеҲ—еҗҚпјҢи®°еҪ•пјҢж•°жҚ®зұ»еһӢпјҢеҲ—еұһжҖ§е’ҢзәҰжқҹ

SQLиҜӯеҸҘзҡ„зұ»еһӢ

  • DDLпјҡж•°жҚ®е®ҡд№үиҜӯиЁҖ data definition language
  • DCLпјҡж•°жҚ®жҺ§еҲ¶иҜӯиЁҖ data control language
  • DMLпјҡж•°жҚ®ж“ҚдҪңиҜӯиЁҖ data manipulation language
  • DQLпјҡж•°жҚ®жҹҘиҜўиҜӯиЁҖ data query language

дҪңз”Ёпјҡ

  • жҺ§еҲ¶ж•°жҚ®зҡ„и§„иҢғжҖ§пјҢи®©ж•°жҚ®жңүе…·дҪ“еҗ«д№үпјҢеңЁеҲ—дёҠиҝӣиЎҢжҺ§еҲ¶

еӯ—з¬ҰдёІ
  • char(32):
    • е®ҡй•ҝй•ҝеәҰдёә32зҡ„еӯ—з¬ҰдёІгҖӮ
    • еӯҳеӮЁж•°жҚ®ж—¶пјҢдёҖж¬ЎжҖ§жҸҗдҫӣ32еӯ—з¬Ұй•ҝеәҰзҡ„еӯҳеӮЁз©әй—ҙпјҢеӯҳдёҚж»ЎпјҢз”Ёз©әж јеЎ«е……гҖӮ
  • varchar(32):
    • еҸҜеҸҳй•ҝеәҰзҡ„еӯ—з¬ҰдёІзұ»еһӢгҖӮ
    • еӯҳж•°жҚ®ж—¶пјҢйҰ–е…ҲиҝӣиЎҢеӯ—з¬ҰдёІй•ҝеәҰеҲӨж–ӯпјҢжҢүйңҖеҲҶй…ҚеӯҳеӮЁз©әй—ҙ
    • дјҡеҚ•зӢ¬еҚ з”ЁдёҖдёӘеӯ—иҠӮжқҘи®°еҪ•жӯӨж¬Ўзҡ„еӯ—з¬Ұй•ҝеәҰгҖӮи¶…иҝҮ255д№ӢеҗҺпјҢйңҖиҰҒдёӨдёӘеӯ—иҠӮй•ҝеәҰи®°еҪ•еӯ—з¬Ұй•ҝеәҰгҖӮ

  • char е’Ңvarcharзҡ„еҢәеҲ«пјҹ

    • е®ҡй•ҝпјҲеӣәе®ҡеӯҳеӮЁз©әй—ҙпјү еҸҳй•ҝпјҲжҢүйңҖпјү
  • charе’Ңvarchar еҰӮдҪ•йҖүжӢ©пјҹ
    • charзұ»еһӢпјҢеӣәе®ҡй•ҝеәҰзҡ„еӯ—з¬ҰдёІеҲ—пјҢжҜ”еҰӮжүӢжңәеҸ·пјҢиә«д»ҪиҜҒеҸ·пјҢй“¶иЎҢеҚЎеҸ·пјҢжҖ§еҲ«зӯү
    • varcharзұ»еһӢпјҢдёҚзЎ®е®ҡй•ҝеәҰзҡ„еӯ—з¬ҰдёІпјҢеҸҜд»ҘдҪҝз”ЁгҖӮ
